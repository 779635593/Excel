# Excel

#基于PhpSpreadsheet做的导出导入Excel类

- 教程：

        1.安装：
            composer require phpoffice/phpspreadsheet
        2.使用：
            导出Excel表格： PhpSpreadsheet::exportExcel()
            导入Excel数据： PhpSpreadsheet::importExcel()
